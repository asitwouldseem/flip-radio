# flip-radio
A project to convert an old Philips RS-100 clock radio into a modern smart speaker.

## Technology
- Raspberry Pi Zero W
- DS3231 Real Time Clock Module
- Adafruit MAX98357 I<sup>2</sup>S Amplifier
- ER-OLEDM028-1W SPI OLED Display
